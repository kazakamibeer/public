## LINE・スプレッドシート連携アプリのコード（JavaScript, GAS）
前職で作ったLINE・スプレッドシート連携アプリのコードです。
LINEでの入力により現場画像付きの見積書をすばやく作成し、PDFファイルとして取引先に送付することができます。
アプリのために作成した業務用LINEアカウント（公式アカウントと呼ばれています）とユーザーが友達になり、対話形式で求められる情報を入力していきます。
